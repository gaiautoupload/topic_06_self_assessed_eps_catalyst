# Topic 06: 自結 EPS 股價催化效應

## 目標

找出公司公布「自結 EPS」或「自結獲利」後，股價是否出現可重複的催化反應，並把事件、估值與交易結果整理成可回測、可展示的資料管線。

## 核心問題

1. 哪些公告屬於自結 EPS 事件？
2. 公告當下的估值是否偏低或具有重估空間？
3. 公告後的短中期股價是否有統計上的正向反應？

## 事件定義

主事件為公告中明確出現自結 EPS、自結獲利或可由公告內容推算 EPS 的公告。

### 優先納入

- 標題或內容包含 `自結`
- 標題或內容包含 `EPS`、`每股盈餘`
- 標題或內容包含 `稅後純益`，且可推算 EPS
- 公司正式公告或法說會文件中的自結獲利資訊

### 暫時排除

- 純新聞評論
- 只有營收、沒有獲利資訊
- 只有股利、減資、增資，沒有自結語境
- 不含數字或無法解析的模糊敘述

## 回測邏輯

1. 以公告日作為事件日。
2. 若公告時間晚於收盤，進場日使用下一交易日。
3. 先做固定持有期回測：
   - `T+5`
   - `T+10`
   - `T+20`
4. 之後再加上停利、停損與估值條件。

## 分析重點

- 事件數量與覆蓋率
- 自結 EPS 的平均報酬
- 不同訊號強度的表現差異
- 不同產業或市場別的差異
- 價格反應的持有期敏感度
